> Еженедельная уборка: зависимости, мёртвый код, синхронизация доков, протухшие задачи.
> _Weekly housekeeping._

## Что делает · Steps
1. `npm audit` / устаревшие зависимости → задача на обновление (без авто-мажор-апгрейдов).
2. Мёртвый код / неиспользуемые экспорты → список кандидатов на удаление (не удалять молча).
3. Доки vs код: `CLAUDE.md`/`HARD-RULES.md`/`tz.md` всё ещё верны? Расхождения → задача.
4. `docs/tasks/in_progress/` залежалось > N дней → вернуть в `ready/` или закрыть.
5. Архивировать старые записи `docs/feedback-journal.md`.

## Границы
- Предлагает изменения задачами; разрушительное (удаление) — только через approval gate.
